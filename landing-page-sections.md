## Implementation Notes

### Global Design Principles (Monzo Framework)
1. **Straightforward Kindness:** No jargon, clear and helpful communication
2. **Simplicity > Cleverness:** Direct messaging that's easy to understand
3. **Friction Reduction:** Just email required, no complex forms or barriers

### Accessibility Requirements
- All icons must have appropriate `aria-label` attributes
- Color contrast must meet WCAG AA standards minimum
- Keyboard navigation support for all interactive elements
- Screen reader friendly semantic HTML structure

### Responsive Behavior
- Mobile-first approach
- Touch-friendly button sizes (minimum 44x44px)
- Readable font sizes on all devices
- Appropriate spacing and padding at all breakpoints

### Technical Requirements
- Icons to be sourced from Lucide or Radix libraries
- Icons should support both light and dark themes
- Consistent sizing across similar elements
- Performance-optimized images and assets
