3. Bash installation instructions:
```markdown
## Bash Completion Installation

### Automatic Installation
```bash
# Add to your ~/.bashrc or ~/.bash_profile
echo 'source ~/.claude/completions/claude-completion.bash' >> ~/.bashrc
source ~/.bashrc
```

### Manual Installation
1. Copy the completion script to your bash completion directory:
```bash
cp ~/.claude/completions/claude-completion.bash /etc/bash_completion.d/
# or
cp ~/.claude/completions/claude-completion.bash ~/.local/share/bash-completion/completions/claude
```
2. Restart your shell or source the completion file
```

4. Zsh installation instructions:
```markdown
## Zsh Completion Installation

### Automatic Installation
```zsh
# Add to your ~/.zshrc
echo 'fpath=(~/.claude/completions $fpath)' >> ~/.zshrc
echo 'autoload -Uz compinit && compinit' >> ~/.zshrc
source ~/.zshrc
```

### Manual Installation
1. Copy the completion script to a directory in your fpath:
```zsh
cp ~/.claude/completions/claude-completion.zsh ~/.zsh/completions/_claude
# or
sudo cp ~/.claude/completions/claude-completion.zsh /usr/local/share/zsh/site-functions/_claude
```
2. Restart your shell or run `autoload -Uz compinit && compinit`
```
