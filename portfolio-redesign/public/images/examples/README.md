## Image Specifications

- Format: JPG (optimized) or WebP
- Max width: 1920px for hero images, 800px for products
- Aspect ratios:
  - Hero: 16:9 or 21:9
  - Products: 1:1 (square)
  - Lifestyle: 4:3 or 16:9
